關於嘉市體（Chiayi City Font）


＊ 規格說明：
	1. 字重：Bold ，中文可搭配思源黑體 Bold 使用效果最佳。
	2. 字符數量：585字。
	3. 支援語言：英文、台客語羅馬拼音、越南文。
	4. 檔案格式：truetype 。
	5. 可於各大作業系統安裝，包含 Windows / Mac / Chrome OS 等，安裝後可在系統軟體中使用，部分軟體之字體選單中可能顯示為英文（ChiayiCity）。
